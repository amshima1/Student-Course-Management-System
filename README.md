# 🎓 Student Course Management System

A Java console application that allows students to manage courses efficiently.

## ✨ Features

- ➕ Add Course
- 📚 View All Courses
- 🔍 Search Course
- 🧮 Compute Total Course Units
- 💾 Save Courses to File
- 📂 Load Courses from File
- 🔁 Recursive Course Counter
- ✅ Duplicate Course Detection

## 🛠 Technologies Used

- Java
- Maven
- Object-Oriented Programming (OOP)
- ArrayList
- File Handling
- Recursion

## 📂 Project Structure

```
src/
 └── main/
      └── java/
           ├── Main.java
           └── Course.java
```

## 🚀 How to Run

1. Clone the repository.

```
git clone https://github.com/amshima1/Student-Course-Management-System.git
```

2. Open in any Java IDE.

3. Run:

```
mvn compile
mvn exec:java
```
